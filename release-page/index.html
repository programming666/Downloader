Downloader

A fast, multithreaded download manager for Windows — built on Qt 6 & MinGW.

Current release: 0.1.0 · Release notes

Download for Windows Source on GitHub

Features

  • Multithreaded downloads

    Split any HTTP/HTTPS resource into N range-requests and merge at the end. Configurable thread count.

  • Resume on failure

    Interrupted downloads pick up where they left off — Range-aware servers send only the missing bytes.

  • Browser hand-off

    Chrome / Edge Manifest V3 extension redirects downloads to a local HTTP endpoint and pre-fills the new-task dialog.

  • Tray & scheduling

    Minimize to system tray; queue timed downloads that fire on a schedule.

  • Dark / light themes

    Switch from the menu — QSS-driven, no restart required.

  • EN / 中文 UI

    Full Chinese and English translations, switched at runtime.

Install on Windows

  1. Download the installer above (DownloaderInstaller.exe).
  2. Right-click → Run as administrator if Windows prompts.
  3. Pick an install folder (default: %USERPROFILE%\Downloader).
  4. Launch from the Start menu or the desktop shortcut.

The installer is produced by the Qt Installer Framework. If Windows SmartScreen blocks the binary, choose More info → Run anyway — the project is open-source and reproducible from source.

Copy install command

winget install --id=Programming666.Downloader -e
